# Daily Marketing Briefings

The morning marketing agent writes one briefing file per weekday into this folder. File naming: `YYYY-MM-DD-briefing.md`. Open the most recent one with your coffee. It contains:

1. **Today's hook** — the strongest content angle for today, with one specific event from HoC's overnight activity (or a fallback case-study reference if no overnight stats are available)
2. **Social drafts** — one LinkedIn post + one X post, both ready to copy-paste
3. **Cold emails drafted** — 3 personalized cold emails saved as Gmail Drafts in your inbox (john@lotlinkin.com), one paragraph summary per prospect
4. **Warm rolodex nudges** — which dealer-friends you haven't touched lately + suggested text/call message
5. **Tomorrow's pre-plan** — one thing to set up tonight to make tomorrow's marketing motion easier

The agent runs weekdays at 6:30 AM local time. If your Mac is asleep or the Claude app is closed, the task fires when you next launch the app. So skipping a day is recoverable — open the app and the briefing catches up.

This folder is the canonical record of every briefing. Don't delete old ones — they're useful for spotting what content angles worked and which dealers you've already touched.
